Prettier

What is prettier?

  • An opinionated code formatter
  • Supports many languages (also *.ts, *.jsx)
  • Works with most editors
  • Very few options

Why?

  • Save and done. Your code is formatted! (VSCode Setting: formatOnSave)
  • ... in a consistent way!
  • No need for you to manually format code -> save time!
  • No formatting discussion in code reviews or in general

Difference to linters?

  • Commonality -> Linters can also have formatting rules!
  • Difference -> Linters can also help to insure Code Quality
  • For Example: no unused variable rule or 'let' and 'const' instead of 'var'
  • Use them together?
